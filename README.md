# 南国市 土曜市 ホームページ

高知県南国市にある土曜だけ開かれる市場の紹介サイトです。昭和の空気感を残しながら、今日の特売、店舗紹介、イベント、ショート動画、お仲間登録までを1ページにまとめています。

## できること

- 今日の特売を切り替えて見せる
- 店舗一覧をカテゴリで絞り込む
- イベント情報をカードで表示する
- ショート動画の導線を置く
- お仲間登録フォームで情報送信の雰囲気を作る
- スマートフォンでも見やすいレスポンシブ表示

## 使い方

ブラウザで [index.html](index.html) を開くと表示できます。

## 更新しやすい箇所

- 特売商品: [app.js](app.js)
- 店舗一覧: [app.js](app.js)
- 見た目: [styles.css](styles.css)
- ヒーロー画像: `assets/hero-image.jpg` に画像を配置

## 補足

実際の動画ファイルや送信先APIはまだ接続していません。公開時は、動画埋め込み先とフォーム送信先をつなぐだけで運用に移れます。
